# ⚽ WorldOps 2026

Cloud-native World Cup 2026 voting platform built to demonstrate real DevOps, Kubernetes, Observability and GitOps workflows.

## Stack

- React + Vite
- FastAPI
- PostgreSQL
- Redis
- Docker
- Kubernetes
- Prometheus
- Grafana
- Loki
- ArgoCD
- GitHub Actions

## Goals

- Real-time voting
- Observability
- GitOps workflows
- CI/CD pipelines
- Kubernetes scaling
- Cloud deployment on AWS
